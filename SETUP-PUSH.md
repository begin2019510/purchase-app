# 📲 手机使用 + 推送通知 配置指南

## 手机访问

采购管理工具是 PWA（渐进式 Web App），手机直接访问即可：

1. 手机浏览器打开 **http://121212121.top**
2. **iPhone Safari**：点底部「分享」→「添加到主屏幕」
3. **Android Chrome**：点菜单→「添加到主屏幕」或「安装应用」
4. 添加后就像一个 App 一样使用

## 推送通知配置

推送使用**飞书自定义机器人**，需要以下步骤：

### 1. 创建飞书自定义机器人

1. 打开飞书，进入一个群聊（或创建一个新群）
2. 点击群设置 → **群机器人** → **添加机器人**
3. 选择 **自定义机器人**
4. 给机器人起个名字，比如「采购管家」
5. 复制 **Webhook 地址**（格式类似 `https://open.feishu.cn/open-apis/bot/v2/hook/xxx`）

### 2. 在 Cloudflare 设置环境变量

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com)
2. 进入你的 Pages 项目
3. **Settings** → **Environment variables**
4. 添加新变量：
   - **Variable name**: `FEISHU_BOT_WEBHOOK`
   - **Value**: 上面复制的 Webhook 地址
5. 保存

### 3. 重新部署

设置完环境变量后，Cloudflare Pages 会自动重新部署。

### 4. 手机端开启推送

1. 手机打开采购管家
2. 点击右上角 **🔔 开启推送**
3. 允许浏览器通知权限
4. 完成！之后新增/编辑/删除采购都会收到飞书通知

## 推送触发场景

| 操作 | 推送标题 | 内容 |
|------|---------|------|
| 新增采购 | 📦 新增采购 | 商品名、平台、价格 |
| 编辑采购 | ✏️ 采购已更新 | 变更内容 |
| 删除采购 | 🗑️ 已删除 | 被删商品 |
| 批量改状态 | 📋 批量更新 | 数量和新状态 |
| 批量删除 | 🗑️ 批量删除 | 删除数量 |
| 新增记账 | 💰 新记录账 | 金额和分类 |
| 编辑记账 | ✏️ 记账已更新 | 变更内容 |
| 删除记账 | 🗑️ 记账已删除 | 金额和分类 |

## 关闭推送

在应用内点击 **🔕 关闭推送** 即可。
